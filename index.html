Tkinter Project

This is my final project in computer engineering
Basicly, this project functions to monitoring the entry and exit of factory employees through the process of recording entrance gate detection

  • Computer Vision
  • Detection Employee
  • Database Management

The first main feature is recording the entry and exit of factory employees. The recording process is based on the success of employees in carrying out the detection process at the factory entrance. If successful, the employee will be considered present and the same will be true during the process of leaving the factory. Then the second feature is adding employee accounts to the factory. This account is used as factory data which will be used as a condition for entering the factory. This account store employee data include employee name, role, employee ID and detailed information history in out the employee factory.
